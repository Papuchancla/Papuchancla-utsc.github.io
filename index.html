Experiencia en Proyectos

Proyecto 1: App movil VulScanner

Este proyecto lo cree con el fin de tener a la mano una herramienta de escaneo de redes provisional para hacer escaneos rapido capaz de mostrar las vulnerabilidades y puertos abiertos en una direccion ip, Esto con el fin de prevenir cualquier ataque por medio de los puertos de los dispositivos conectados a la red. Las tecnologias que use fueron varias librerias para el uso de la red desde los dispostivos moviles además de las tecnologias del IDE de Android Studio para poder crear esta aplicación móvil.

Proyecto 2: Programa de Python para hacer ataques ARP Spoofing

Para este proyecto hice una herramienta que me permite hacer ataques ARP Spoofing para saturar la red de una direccion ip en especifico el script lo diseñe para tomar una ip de alguna maquina que este, conectada a la red para usarla y mandar paquetes ARP que saturaran el trafico de paquetes de la maquina victima esta herramienta la cree para acompañarla con otras herramientas que me permitan analizar las redes. Las tecnologias que utilize fueron Flask, ademas de librerias que me permitieron usar los componentes de red de mi pc para mandar los paquetes ARP.

Proyecto 3: Pagina Web para viajes de autobuses

Este fue un proyecto de equipo donde me toco apoyar a los del frontend para crear un diseño llamativo y amigable para el usuario mientras el backend de encargaba de las funcionalidades de la pagina. Las herramientasutilizadas fueron Angular y node.js.

Proyecto 4: Pagina web de ATO

Participe en un proyecto de una pagina web de ATO para promocionar sus productos a demas de que desde la pagina se pudo descargar la app movil que cree para ellos. se uso HTML, css y JS vanilla para crear la pagina ademas de usar Firebase para la base de datos.

Proyecto 5: ATO Apprecia

Eta fue una app movil con el fin de premiar a los empleados de la empresa ATO por su desempeño. Esta app fue desarrollada en Android Studio se usaron tecnologias del IDE a demas de la base de datos Firebase para guardar imagenes y los premios a demas de el registro de los empleados y las calificaciones de cada uno.
